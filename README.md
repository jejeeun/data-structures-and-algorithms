# Data Structures and Algorithms

## 구현한 자료구조

```
┌──────────────────────┬──────────────────────────────────────────────────────┬─────────────────────────┐
│ 자료구조              │ 한줄 요약                                            │ 파일                     │
├──────────────────────┼──────────────────────────────────────────────────────┼─────────────────────────┤
│ Linked List          │ 노드가 다음 노드를 가리키며 한 방향으로 연결된 구조       │ LinkedList.mjs          │
├──────────────────────┼──────────────────────────────────────────────────────┼─────────────────────────┤
│ Doubly Linked List   │ 노드가 앞/뒤 양방향으로 연결되어 양쪽 탐색이 가능한 구조  │ DoublyLinkedList.mjs    │
├──────────────────────┼──────────────────────────────────────────────────────┼─────────────────────────┤
│ Stack                │ 가장 나중에 넣은 데이터가 먼저 나오는 구조 (LIFO)        │ Stack.mjs               │
├──────────────────────┼──────────────────────────────────────────────────────┼─────────────────────────┤
│ Queue                │ 가장 먼저 넣은 데이터가 먼저 나오는 구조 (FIFO)          │ Queue.mjs               │
├──────────────────────┼──────────────────────────────────────────────────────┼─────────────────────────┤
│ Deque                │ 양쪽 끝에서 삽입/삭제가 모두 가능한 구조                 │ Deque.mjs               │
├──────────────────────┼──────────────────────────────────────────────────────┼─────────────────────────┤
│ Hash Table           │ 키를 해시 함수로 변환해 O(1) 접근이 가능한 구조          │ HashTable.mjs           │
├──────────────────────┼──────────────────────────────────────────────────────┼─────────────────────────┤
│ Hash Set             │ 해시 테이블 기반으로 중복 없이 데이터를 저장하는 구조      │ HashSet.mjs             │
└──────────────────────┴──────────────────────────────────────────────────────┴─────────────────────────┘
```
